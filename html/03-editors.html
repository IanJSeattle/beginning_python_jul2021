Python-friendly Editors

Choose a text editor, not a word processor.

Nice features to have:

  • Syntax highlighting
  • Auto-indent
  • Quote/paren/bracket matching
  • Code-aware navigation
  • Debugger/linter/interpreter integration

Some Options

Plain text editors:

  • Notepad++ (Win)
  • Sublime Text
  • EMACS
  • vim

Dedicated IDEs:

  • Visual Studio Code
  • Thonny
  • Atom
  • PyCharm
  • Eclipse

What we recommend

  • Sublime Text
  • Visual Studio Code

This guide from RealPython is also good.

What we use

Ian: vim

Maria: EMACS

These are not good beginner choices, unless you already use them.
