The Python interpreter

We will use the interative interpreter to demonstrate ideas in class. Start the interpreter by typing 

python
in your terminal. Enter 
quit()
 to exit the interpreter.

Commands entered in the interpreter are executed immediately.

The interpreter does one extra thing in interactive mode: if you enter a value, it will interpret the value and print it out.

Interactive samples will show the >>> prompt.

File Mode

The way Python is usually used is to run files. Python files use the .py extension, like hello_world.py. We'll start using this technique shortly.

To use Python in this way, type this at your computer's command prompt: 

python hello_world.py
