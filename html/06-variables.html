What is a variable?

  • A symbol representing something that can be changed
  • Specifically, a label pointing to an object
  • Not a "container" for a value

A simple variable 

>>> suit = "hearts"
>>> print(suit)
hearts

Variable names

Tech specs:

  • Upper case, lower case, underbar, numbers
  • Case sensitive
  • Leading numbers not allowed
  • Spaces not allowed
  • Length unlimited

Style considerations

Keep the following in mind:

  • snake_case preferred over camelCase
  • Stick with lower-case letters for now
  • Be descriptive
  • Don't be verbose

Examples

Say we want a variable to hold a playing card.

  • Wrong capitalization: Card
  • Too terse: c
  • Too verbose: current_playing_card
  • Not snake_case: PlayingCard
  • Just right: card

Objects in Python

Everything in Python is an "object." A variable takes on the characteristics of the object it points to. A variable can contain any object.

Data Types

Python recognizes many data types, and more can be created. Variables are not limited in which types they can take. Python variables are not "typed."

Type Checking

Python checks types at runtime. There is an optional system for compile-time type checking.
