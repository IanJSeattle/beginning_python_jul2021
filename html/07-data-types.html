Data Types in Python

Simple Data Types

  • int (Integer)
  • float (Floating point)
  • str (String)

Complex Data Types

  • dict (Associative Array)
  • list (List)
  • tuple (Immutable List)

Integers

  • Whole numbers (no decimal point)
  • 0, 25, -300, and 1000000 are all integers
  • Don't mix nicely with strings
  • Do mix nicely with floating point numbers

Floating Point Numbers

  • Have decimal points
  • 1.0, 3.141592654, -17.25, and 1000000.00001 are all floats
  • Don't mix nicely with strings
  • Do mix nicely with ints
  • Basic float math is inherently imprecise 

>>> 17.25 + 18.00003
35.250029999999995

Strings

  • A string is zero or more characters
  • The empty string, "", is still a string
  • Use either single quotes, ', or double quotes, "
  • Don't mix nicely with ints and floats
  • Interesting methods, like .upper(), .lower(), .capitalize()
