Complex Data Types

This is just a brief introduction. More to come later.

Lists

  • Constructors: [] or list()
  • Contains a list of any type of object
  • Mutable
  • Random-access
  • Indexed by number
  • 0 (zero) is first element 

>>> my_list = ['one', 2, 'trois']
>>> my_list[1]
2

Tuples

  • Constructors: () or tuple()
  • Contains a list of any type of object
  • Immutable
  • Random-access
  • Indexed by number
  • 0 (zero) is first element 

>>> my_tuple = ('one', 2, 'trois')
>>> my_tuple[1]
2

Dicts

  • Constructors: {}* or dict()
  • Contains a collection of any type of object
  • Mutable
  • Random-access
  • Indexed by immutable keys
  • Fast 

>>> my_dict = {'value': 2, 'suit': 'hearts'}
>>> my_dict['value']
2

* Note that {} can also be used to construct a set.

Mutability in Python

mutability adj. Capable of or subject to change or alteration.

An object that's mutable can be changed or updated. An object that's immutable cannot be changed.

Strings, integers, and tuples are immutable. Lists, dicts, and objects are mutable.
