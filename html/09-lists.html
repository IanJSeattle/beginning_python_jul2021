Lists in Depth

  • Constructors: [] or list()
  • Contains a list of any type of object
  • Mutable
  • Random-access
  • Indexed by number
  • 0 (zero) is first element 

>>> my_list = ['one', 2, 'trois']
>>> my_list[1]
2

What is a Constructor?

A constructor is a thing which "constructs" a new object for you. You can use [] or list() to construct a list object, like so: 

my_list = [1, 'two', 3]

my_list = list(1, 'two', 3)

These create a list object, which contains an integer, a string, and another integer, in that order.

Lists Can Contain Anything

A list can hold numbers, strings, other lists; any object (and everything is an object) can be stored in a list.

A list which contains other lists is so common it has its own name: a multi-dimensional list.

Lists are in Order

Lists preserve the order in which objects are added.

Accessing List Contents

Use square brackets to access a particular item in a list: 

>>> my_list = ['maria', 'ian', 'sarah']
>>> print(my_list[2])
sarah

Lists are Mutable 

>>> my_list = [1, 'two', 3]
>>> my_list.append('four')
>>> my_list
[1, 'two', 3, 'four']
>>> my_list.insert(0, 'zero')
>>> my_list
['zero', 1, 'two', 3, 'four']
>>> del my_list[4]
>>> my_list
['zero', 1, 'two', 3]
>>> my_list[1] = 'one'
>>> my_list
['zero', 'one', 'two', 3]
