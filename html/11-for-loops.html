Loops

Python includes two loop styles:

  • For loop
  • While loop

Let's dicuss what a loop is, first.

What is a loop?

In programming, a loop is a mechanism to allow a set of actions to be performed over and over, in a controllable manner. 

names = ['Bob', 'Sue', 'Jim', 'Ann']
print(names[0])
print(names[1])
print(names[2])
print(names[3])

Versus: 

names = ['Bob', 'Sue', 'Jim', 'Ann']
for name in names:
    print(name)

A Brief Diversion on Indentation in Python

Python uses whitespace to define blocks of code: 

names = ['Bob', 'Sue', 'Jim', 'Ann']
for name in names:
    print(name)
  • Line 2 defines the terms of the loop.
  • Line 3 is the indented block of code that is executed as a loop.
  • A colon is always* followed by an indented block of code.

* Almost always

Python will recognize both tabs and spaces and whitespace. PEP8, the Python style guide, says:

  • Use spaces
  • 4 spaces is the standard indentation

And Now, Back to Loops

The definition of a for loop in Python:

For each item in an iterable object, run the following code once, and set the specified variable to that item. End the loop when the iterable is exhausted.

Looping Over Numbers

Common task: loop N number of times.

range() to the rescue! 

for num in range(10):
    print(num)

Numbering Items While Looping

Common task: know the number of an item while looping.

enumerate() to the rescue! 

for num, name in enumerate(names):
    print(f'Name number {num+1} is {name.title()}')

List Comprehensions

A list can be created with a shorthand syntax called a comprehension. This code creates a list of squares: 

squares = [num*num for num in range(10)]

Can you spot the for loop in the comprehension?

Equivalent for loop: 

squares = []
for num in range(10):
    squares.append(num*num)
