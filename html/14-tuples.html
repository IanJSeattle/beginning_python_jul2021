Tuples

A tuple (short for "multiple") is a lot like a list, with one key difference: it can't be modified. 

>>> my_list = [1, 2, 3, 4]
>>> my_list[0] = 7
>>> my_list
[7, 2, 3, 4]
>>> my_tuple = (1, 2, 3, 4)
>>> my_tuple[0] = 7
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'tuple' object does not support item assignment

Syntax

Tuples are created with parentheses: 

>>> my_tuple = (1, 2, 3, 4)

Tuple elements are accessed with square-bracket indexes, like lists: 

>>> my_tuple[2]
3

Looping over a tuple

Just like a list, you can loop over a tuple: 

>>> my_tuple = ('two', 'three', 'four', 'five')
>>> for item in my_tuple:
...     item = item.title()
...     print(f'{item}! {item} tuples, ah! ah! ah! <thunder>')
...
Two! Two tuples, ah! ah! ah! <thunder>
Three! Three tuples, ah! ah! ah! <thunder>
Four! Four tuples, ah! ah! ah! <thunder>
Five! Five tuples, ah! ah! ah! <thunder>
