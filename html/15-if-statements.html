If Statements

The if statement is used to make decisions in code. It does this using a boolean data type.

Booleans

The boolean data type is one which has only two values: true and false. In Python, these are known as True and False (note capitalization). The data type is called bool.

Any value in Python can be cast (that is, transformed) into a bool.

Truthiness

Something which is "truthy" evaluates to True when cast to bool:

  • 'hello' (non-empty string)
  • 1 (non-zero integer or float)
  • [1] (non-empty list)
  • (1) (non-empty tuple)
  • {'one': 1} (non-empty dict)

Falsiness

Likewise, something which is "falsy" evaluates to False:

  • '' (empty string)
  • 0 (zero-value int or float)
  • [] (empty list)
  • () (empty tuple)
  • {} (empty dict)

Booleans in Action

Start up your interactive Python session, and try running the following:

  • bool(1)
  • bool(0)
  • bool([1, 2, 3])
  • bool('hi there')
  • bool('')

Checking Equality

The == (double-equals) operator returns a boolean value. The = (equals) operator assigns values, but doesn't return anything.

Use == to check if two things are equal*.

* There are a lot of caveats to this when discussing complex data types.

Use == to compare simple values, such as numbers or strings, or single elements of more-complex data types.

Do not use it to compare whole lists or dictionaries until you have a better understanding of how those types work.

Structure of an If 

if boolean:
    # perform actions in the true block
elif boolean:
    # perform actions in the else-if block
else:
    # perform actions in the else block
