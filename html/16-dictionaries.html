Dictionaries

A dict object is like a list, but instead of using numbers for the index, it can use any immutable value -- a string, a floating point number, a tuple, etc.

my_dict = {'name': 'Ian', 'role': 'instructor}

Useful as a mini database inside a program, among other things.

Â dicts can be instantiated with either the dict() keyword, or using {} curly brackets.

Using {} brackets is slightly more efficient.

Just like a list, you can iterate over a dict: 

>>> my_dict = {'name': 'Ian', 'role': 'instructor'}
>>> for item in my_dict:
...   print(item)
...
name
role

Note that this prints the keys in the dict, not the values.

Interesting iteration methods on dicts:

  • .keys(): return just keys (default)
  • .values(): return just values
  • .items(): return both key and value as a tuple

A very useful way to loop over dicts: 

>>> my_dict = {'name': 'Ian', 'role': 'instructor'}
>>> for key, value in my_dict.items():
...   print(f'{key}: {value}')
...
name: Ian
role: instructor

Deleting Items

dicts (and lists) allow you to easily remove objects, using the del keyword: 

>>> my_dict
{'name': 'Ian', 'role': 'instructor'}
>>> del my_dict['role']
>>> my_dict
{'name': 'Ian'}

Nesting dicts

A list or dict can contain any Python object as its value, and this includes other lists or dicts. For instance, this list of dicts is a valid, and commonly used construction: 

>>> my_hand = [{'value': 2, 'suit': 'hearts'},
...            {'value': 5, 'suit': 'spades'}]
>>> my_hand[0]
{'value': 2, 'suit': 'hearts'}
