Classes in Python

An Introduction

What is a Class?

Classes are used to represent things. A class is a data type which has data (attributes) and actions (methods) associated with it.

For instance, you might have a Car class, which would have a fuel_level attribute, and accelerator() and brake() methods.

A Simple Class 

class Car:
    def __init__(self):
        self.fuel_level = 0
        self._accelerator_position = 0
        self._brake_position = 0

    def accelerator(self, position):
        self._accelerator_position = position

    def brake(self, position):
        self._brake_position = position

Classes Create Objects

Think of a class like a blueprint: it shows how a thing is put together, but is not the actual thing that's built.

The actual thing that's built is the object. In Python, you instantiate (that is, build) an object by calling a class like so: 

my_car = Car()

Now, my_car is an object, which is an instance of Car.

Object Attributes

An attribute is a variable, which is attached to an object. In our example so far, fuel_level is the only public variable.

Python doesn't have a concept of variable privacy, so by convention, any variable starting with an underbar (like _accelerator_level) is considered private.

Object Methods

A method is a function, which is attached to an object. The magic of object methods is they have internal access to all the variables of the object.

Imagine a PlayingCard object. It would store its value (2-10, J, Q, K, A), and its suit as attributes. You could have a points() method that reports its point value.

Now, calling jack_of_hearts.points() will return 10, instead of 'J', making the math easy.
